# buoy_sonar_python

Trajectory description with different object types

Human:
- linear with cyclic disturbances
- speed = 0...2 

Dolphin:
- parametric nonlinear trajectory
- speed <= 10 

Drone:
- linear
- speed = 5...30

Linear trajectory parameters:
- x_start
- y_start
- z_start
- x_speed
- y_speed
- z_speed
